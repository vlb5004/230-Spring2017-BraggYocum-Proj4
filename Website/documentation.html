Project

Throughout the assignment we wanted to create an immersive full page experience, so we learned how to use Fullpage.js plugin to create this. Each wonder has it’s own page with information in the icons when you hover over it and a detailed illustrations. Also, we used advanced javascript to animate the Wonders on each page and icons to pop-up information on each page.

Above & Beyond

We used topics outside that were not discussed in class such as Fullpage.js and other javascript functions to animate the Wonders. Also, we created custom illustrations and unique interface details compared to using a bootstrap or template.
